Python for Beginners:

Lesson 1 Hello World!

Hi, welcome to another lesson on Python for Beginners.

As we went over how to set up an environment for Python, today, we will start writing some code.

There is a number 1 step you need to do to start learning any programming language you want. That first step is to make a Hello World program. This is nothing too advanced or difficult, but you are simply showing some text on your command line(usually).

So, let's open our Visual Studio Code.

You will see soething like this.

First of all, we will create a folder of any name with any path. It could be in your Documents, Desktop, or anywhere you want.

I have created a folder called Python Lessons on my Desktop.


After that, click the Open Folder button to open the folder you have created.


After that, you will be able to see a screen like this. In my case, because I named my folder Python Lessons, you can see “Python Lessons” on the left.

But, as you can notice, there is nothing there. So we will create a new file.

Click the button with the red rectangle to create a new file.

Then, you will see a blank section where you can enter the name of your file. Here you could name anything, but we will call it “Lesson_1.py”. It is important to put that “.py” at the end because it indicates that the file we are making is a python file. I recommend just putting the underscore “_” instead of using the space bar because it will be more convenient to deal with files.

Then you see another tab opened up on the main screen. Let’s close the Welcome tab so that we can focus on our first Hello World tab.

So, now we can actually write some python code!

I will write this:

print("Hello World!")

And... that's it! Very simple right?



Let's break it down so that we can understand what is happening.
So, on the first line, we write “print”. This print is like a command that literally shows some texts on the command line. And as you can guess, the texts that show up are the things we write in the parentheses.
So by writing

print(“Hello World!”)

We are telling the computer to show the texts that I give you. And the text, in this case, is Hello World!

After that, don’t forget to save the file by pressing Ctrl + s or just by using your mouse.

Finally, let’s run this program. There are many ways to do this, but we will utilize the tool we have in Visual Studio Code.
By pressing Ctrl + ` you can open up your Terminal or Command line. Or you could open up by clicking the button on top.

Then, at the bottom, you will see a screen like this. Normally, you have to set your path correctly, but since we opened up the folder itself so automatically the Visual Studio Code will set the path for you.

And at the place wehre the arrow is pointing, we will select and write this:
python .\Lesson_1.py

After that just press the Enter button.

And you can see the text that you wrote, in this case, "Hello World!"

YAY! We just successfully wrote our first Python program! Congratulations!

See you later!
