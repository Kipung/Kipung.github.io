Today, we are going to learn about two loops: for loop and while loop.

Loops in general

Whether it is while or for loop, loops help us to write long repeating codes into shorter version that accomplish the same task.

For example, to write hello 5 times might look like this without a loop: 

    

print("hello")
print("hello")
print("hello")
print("hello")
print("hello")

But using a for loop, it looks like this:

    

for x in range(5):
    print("hello")

You can see that loops can be very useful for writing repeating patterns. Don't worry about the exact syntax here, we will go over them soon.

While Loop

First, let's go over while loop.

                


while (True):
    print("Hello")

From this example, we actually get an infinite loop that keeps printing "Hello".

Basically, if the statement or variable is True in the parentheses, then the code under while (with the indentation) will run.



So, in the parentheses it can go with simple logic statement like whether 3 is bigger than 5

    

while (3>5):
    print("hello")

For this example above, the while loop will not run at all because the statement in the parentheses is simply not true.

    
        

while (False):
    print("hello")

Above will also not run because it is using "False" with the capital f, which is a boolean value for false. So for while loops, you can simply use False, or True to run it. However, when using True, it would result in an infinite loop.



However, mostly while loops are used together with incrementation or decrementation.
For example: 

    


x = 0
while (x<5):
    print("hello")
    x = x+1

This example above will print out "hello" 5 times and stop. And probably you can see why it stops after 5 times

Since, both lines are under while loop with the indentation, they are both run. And the x = x+1 line will increase the value of x by 1 which is called incrementation. So as it loops, it reaches 5 where the statement becomes false and the while loop stops looping.



Decrementing loop works as well.

    
        

x = 10
while (x>=1):
    print("hello")
    x = x-1

But here, notice. The less than or equal to sign. Because of this, it will run 10 times. Without the equal sign, it will only run 9 times. If you want to check, you can check by iterating through each line and updating the variable x on a paper or something. Alternative method is that you put 0 instead of 1 and just leave only less than sign, then it will run 10 times as well.



I think that was enough of while loops. I personally think that at the beginning while loops may appear more difficult than for loops. Let's look at for loops now.

For Loop

Let's start with an example.

    


ran_list = [1,2,3,4,5]    
for x in ran_list:
    print(x)

After running this code above, you get this:

    
1 
2 
3 
4 
5

The "x" we put right next to "for" is a variable name. At that moment of running the for loop, we are basically creating a temporary variable that will go through the list we created beforehand:"ran_list". And we call this process an iteration.


As "x" iterates through the list, it takes each value from the list and goes to the code under for loop statement, which in this case we only have "print(x)". So, as it tries to print, it calls the variable "x" that we created temporarily only within this for loop. So, it iterates the first element in the "ran_list", which is 1, so the first line of printing is 1. And the iteration continues printing out 1 through 5. 


    


ran_list = [1,2,3,4,5]
for x in ran_list:
    print("hello")

However, in this example above, we don't even use the variable x that we iterated through the list. This is totally fine. However, since we iterate through the list 5 times, we successfully print "hello" 5 times. So, either way you can do, whether you want to use the variable or not.


A good tip for iterating through a know number of times, actually "range" function is used a lot. Here is the code for that:

    


for x in range(5):
    print("hello")

This code example above, also prints out "hello" 5 times. You can think of "range" as list that starts with 0 and ends at the number you put but that number is exclusive. So, for this case, it starts with 0 but ends at 5 without including 5. So, if we hypothetically create a list it would be something like this: [0,1,2,3,4].


With "range", we can not only loop through the number of times you specify, but also count too.

    


for x in range(5):
    print(x)

This example above will print out 0 to 4. And if you are picky and want to print out 1 through 5, then you could just add 1 to print statement.

    

for x in range(5):
    print(x+1)

With the "+1", you can actually get from 1-5.
